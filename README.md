# Customer Segmentation using K-Means

## Problem Statement
The goal of this project is to segment customers based on their annual income and spending score using K-Means clustering.

## Dataset Link
https://www.kaggle.com/code/hamoulhadjlynafarah/mall-customers-segmentation

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. K-Means Clustering
6. Customer Segmentation

## Results
Successfully grouped customers into different segments based on spending patterns and annual income.

## Conclusion
Customer segmentation helps businesses understand customer behavior and improve marketing strategies.
